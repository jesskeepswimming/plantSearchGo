## To run locally: 

backend:
1. cd server
2. add an .env file to the server directory
```
DB_USER="youruser"
DB_PASSWORD="yourpassword"
DB_HOST="your localhost"
DB_PORT="your database port"
DB_TEST="your test database"
 ``` 
3. nodemon index

frontend:
1. cd client
2. npm start
